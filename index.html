FSM
Fernie Server Mod

FSM (Fernie Server Mod) — комплексный серверный мод для Fabric 1.20.1, превращающий обычный сервер в полноценный ролевой/RPG-проект: экономика, роли, защита территорий, тюрьма, аукцион, казино, встроенный ИИ-помощник по игре (/ai), NPC-магазины и многое другое — всё из коробки, без необходимости ставить десяток отдельных плагинов.
